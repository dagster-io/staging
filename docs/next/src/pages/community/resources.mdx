  The 0.9.0 release **_Laundry Service_** introduces two user facing API changes hooks and configured and enhanced Kubernetes deployment strategy.
  Tamas Nemeth, the tech lead of data platform team with Prezi, demonstrates why and how to move data pipelines of 900 jobs

- [**Jun 2020 Announcing Dagster 0.8.0**](https://www.youtube.com/watch?v=OzF0Vt4BBIo&ab_channel=dagster)
  The 0.8.0 release **_In The Zone_** enables Dagster to become a multi-team platform, introduces the asset manager
  and the Dagster-native orchestration cluster, improves scheduler, re-execution and PySpark support on EMR and Databricks,
  and allows automatic Airflow Dags/DagBag auto-ingestion.
